# Evidence Visuals (FTD)

This repo already contains a large conceptual figure suite generated by:

- `dissemination/manuscript/figures/generate_figures.py`

This file documents an **evidence-focused addendum**: 22 figures that emphasize **numerical stability, residuals, sensitivity, robustness, and look-elsewhere diagnostics**.

## Generate

```bash
python dissemination/manuscript/figures/generate_evidence_figures.py --list
python dissemination/manuscript/figures/generate_evidence_figures.py --all --dpi web --samples 50000
python dissemination/manuscript/figures/generate_evidence_figures.py --all --dpi print --samples 200000
```

Outputs are written as **PNG + SVG** to:

- `dissemination/manuscript/figures/ch14/fig-evidence-*.png`
