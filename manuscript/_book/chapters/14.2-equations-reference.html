70.2 Flux and Density

70.2.1 Flux Vector

\[ \vec{J}(x) = [J_x, J_y, J_z] \]

70.2.2 Density (Flux Magnitude)

\[ \rho(x) = |\vec{J}(x)| = \sqrt{J_x^2 + J_y^2 + J_z^2} \]

70.2.3 Density Gradient

\[ \nabla\rho = \left(\frac{\partial\rho}{\partial x}, \frac{\partial\rho}{\partial y}, \frac{\partial\rho}{\partial z}\right) \]

70.2.4 Discretized Gradient

\[ \nabla\rho(x) = \frac{1}{2}\left(\rho(x+\hat{e}) - \rho(x-\hat{e})\right) \]

70.3 Field Operators

70.3.1 Divergence

\[ \text{div } \vec{J} = \nabla \cdot \vec{J} = \frac{\partial J_x}{\partial x} + \frac{\partial J_y}{\partial y} + \frac{\partial J_z}{\partial z} \]

Discretized: \[ \nabla \cdot \vec{J} = \frac{1}{2}\sum_i \left[J_i(x+\hat{e}_i) - J_i(x-\hat{e}_i)\right] \]

70.3.2 Curl

\[ \text{curl } \vec{J} = \nabla \times \vec{J} = \begin{pmatrix} \frac{\partial J_z}{\partial y} - \frac{\partial J_y}{\partial z} \\ \frac{\partial J_x}{\partial z} - \frac{\partial J_z}{\partial x} \\ \frac{\partial J_y}{\partial x} - \frac{\partial J_x}{\partial y} \end{pmatrix} \]

70.3.3 Laplacian

\[ \nabla^2 f = \frac{\partial^2 f}{\partial x^2} + \frac{\partial^2 f}{\partial y^2} + \frac{\partial^2 f}{\partial z^2} \]

Discretized (3D): \[ \nabla^2 f(x) = \sum_{\text{neighbors}} f(n) - 6f(x) \]

70.4 Wave Equation

70.4.1 Continuous Form

\[ \frac{\partial^2 u}{\partial t^2} = c^2 \nabla^2 u \]

70.4.2 Discrete Update (Velocity-Verlet)

  1. Compute Laplacian: \[ \nabla^2 u = \bar{u}_{neighbors} - u \]

  2. Update wave velocity: \[ v_{wave}(t+1) = v_{wave}(t) + c^2 \nabla^2 u \]

  3. Update flux: \[ \vec{J}(t+1) = \vec{J}(t) + v_{wave}(t+1) \]

  4. Apply damping: \[ \vec{J}(t+1) = \vec{J}(t+1) \times (1 - \lambda) \]
